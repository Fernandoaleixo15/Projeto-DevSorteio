<h1>O DevSorteio</h1>
<h2>O que é o projeto?</h2>
<p>É uma aplicação web simples que permite ao usuário realizar o sorteio de números aleatórios dentro de um intervalo definido. </p>
<hr>
<h2>Principais funcionalidades:</h2>
<ul>
  <li>Entrada de Dados: O usuário insere dois valores (um número mínimo e um número máximo).</li>
  <li>Lógica de Sorteio: Ao clicar em um botão, o sistema gera um número aleatório que esteja obrigatoriamente entre os dois valores informados.</li>
  <li>Exibição do Resultado: O número sorteado é mostrado na tela de forma clara para o usuário. </li>
</ul>
